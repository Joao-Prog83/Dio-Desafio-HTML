O que é HTML?

(Voltar)

É uma linguagem padrão utilizada para criar páginas Web. Em suas primeiras versões, era responsável pela formatação da página, mas com o seu aprimoramento o HTML (HyperText Markup Language) está se tornando cada vez mais um sistema estrutural (responsável pela estrutura da página), deixando essa função de formatação para a linguagem CSS. É fundamental que ao se criar uma página Web, a linguagem HTML precisa ser desenvolvida com uma Semântica adequada, permitindo que os navegadores interpretem e exibam o conteúdo da página corretamente.

Estrutura básica do HTML

(Voltar)

É uma convenção e sempre indicado que a Primeira Página Web do seu site seja denominada com “index.html”.
Para estruturarmos a página usamos as tags:


  • < html > elemento raiz de uma página HTML;

  • < head > elemento contém meta informações sobre a página HTML;

  • < title > elemento que especifica um título da página HTML;

  • < body > elemento que define o corpo do documento, onde iremos colocar todo o conteúdo visível da página HTML como títulos, parágrafos, imagens, hiperlinks, tabelas, listas etc.

Um pouco sobre tags

(Voltar)

Tags <> são elementos de marcação da linguagem, que informa ao navegador como exibir o conteúdo da página. No HTML, as tags <> delimitam e estruturam o conteúdo de modo que o navegador compreenda qual o tipo de elemento indicado na formatação, como um Texto, uma Imagem, um Link, uma Tabela, um Formulário etc.

Conhecendo algumas Tags

(Voltar)

Tags de texto


  • < h1 > elemento título

  • < p > elemento parágrafo

  • < br > elemento de quebra para próxima linha

  • < blockquote > elemento citação

Tags de formatação de texto


  • < i > elemento itálico

  • < strong > elemento negrito

  • < u > elemento underline (sublinhado)

  • < mark > elemento marca texto

  • < sup > elemento superior

  • < sub > elemento inferior

Tags de listas


  • < ul > elemento lista não ordenada

  • < ol > elemento lista ordenada

  • < li > elemento inserção de item em lista ordenada

Tags de link


  • < a href > elemento link

Mídias no HTML

(Voltar)

Tudo que envolve som, vídeo ou imagem são considerado mídias que fornecem elementos a serem inseridos numa página Web. Tags:


  • < img > elemento arquivo de imagem;

  • < audio > elemento usado para inserir um áudio;

  • < video > elemento usado para mostrar um vídeo;

  • < source > elemento que permite especificar arquivos de vídeo e áudio alternativos que o navegador pode escolher. O navegador sempre vai utilizar o primeiro formato reconhecido.

Semântica e sua importância

(Voltar)

Quando falamos em semântica no HTML, estamos nos referindo principalmente a criação de uma estrutura de página Web de modo que o sentido lógico esteja de acordo com o que está proposto a fazer. Isso permite que a NAVEGABILIDADE na página melhore a interação de pessoas com necessidade ou limitações. O desenvolvedor precisa se preparar para lidar com questões de Acessibilidade. O uso da semântica também é fundamental para que robôs de busca consigam acessar e facilitar a indexação da página num site de busca, o que vai auxiliar na ampliação da base de acesso da página.
